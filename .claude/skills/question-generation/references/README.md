# User Question Lists

This directory holds the user's personal due diligence question lists.

## How to Add Your Questions

Upload or paste your question lists as markdown files in this directory. Name them descriptively:

- `initial-screening-questions.md` — Questions for the first call with a broker
- `cim-review-questions.md` — Questions after reviewing a CIM
- `deep-diligence-questions.md` — Questions for detailed due diligence
- `seller-meeting-questions.md` — Questions for face-to-face with the seller

Or use any naming convention that makes sense to you.

## Format

Each file should be a simple list of questions. You can optionally add notes or categories:

```
## Customer Questions
- How did you acquire your current customer base?
- What's your customer retention rate?
- Tell me about your top 5 customers.

## Financial Questions
- Walk me through your add-backs line by line.
- Why is there a difference between your P&L and tax return revenue?
```

The question-generation skill will read these files and incorporate relevant questions into its prioritized output, matching them to the current deal state.
